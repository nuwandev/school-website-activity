# School Website

This project is a website for a school called "Siripura Central College".

**Note:** This website is not responsive. Please refer to the screen recording in the `screen-record` directory to see how it is intended to look.

## Features

- **Multi-page Layout**: The website has several pages, including Home, About Us, Academics, and Contact Us.
- **Hero Section**: The homepage features a hero section with a welcome message and an image.
- **Vision and Mission**: A section dedicated to the school's vision and mission.
- **Summary**: A summary section with key statistics about the school.
- **Recent Activities**: A section to display recent activities or blog posts.

## How to Use

1. Open the `index.html` file in your web browser to view the homepage.
2. Use the navigation bar to visit the other pages:
   - `about-us.html`: Learn more about the school.
   - `academics.html`: View information about the academic programs.
   - `contact.html`: Find contact information.

## Project Structure

- `index.html`: The homepage of the website.
- `about-us.html`: The "About Us" page.
- `academics.html`: The academics page.
- `contact.html`: The contact page.
- `assets/`: This directory contains images, CSS, and JavaScript files.
- `118iCM_IT_School_Website_Design_Assignment.pdf`: The assignment description for the project.
- `screen-record/`: This directory may contain screen recordings of the website.

## Learning Outcomes

- **Multi-page Website**: Created a static website with multiple pages (Home, About Us, Academics, Contact Us).
- **CSS Styling**: Gained experience in styling a website using pure CSS, including layout, typography, colors, and backgrounds.
- **Flexbox**: Used Flexbox to create layouts and align elements.
- **Fixed Navigation**: Implemented a fixed navigation bar that stays at the top of the page.
- **CSS Transitions and Transforms**: Used CSS transitions and transforms to add hover effects and other animations.
- **File and Folder Structure**: Organized the project with separate folders for CSS, images, and other assets.
- **Semantic HTML**: Used semantic HTML tags to structure the content of the website.
